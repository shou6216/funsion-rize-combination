## Usage

1. setup node.js development
2. git clone
3. npm global install
```
> npm install express -g
> npm install express-generator -g
```
4. npm install
```
> cd geed-rizer-pattern
> npm install
```
5. run
```
> cd geed-rizer-pattern
> npm start
```
